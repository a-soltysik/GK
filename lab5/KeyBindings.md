# Key bindings for Lab 5 scene -

- **W** - Move forward
- **S** - Move backward
- **A** - Move left
- **D** - Move right
- **Left Shift** - Move down
- **Space** - Move up


- **0** - Main camera
- **1** - Light 1 camera
- **2** - Light 2 camera
- **3** - Light 3 camera
- **4** - Light 4 camera
- **5** - Light 5 camera
- **6** - Light 6 camera
- **7** - Light 7 camera
- **8** - Light 8 camera


- **,** - Ambient change state
- **.** - Diffuse change state
- **/** - Specular change state
- **L** - Red change state
- **;** - Green change state
- **'** - Blue change state
- **↑** - Value up
- **↓** - Value down


- **Left Ctrl** - Show normal vectors (On/Off)
- **Tab** - Wireframe mode (On/Off)
